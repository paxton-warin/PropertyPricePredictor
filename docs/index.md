# PropertyPredictor Documentation

PropertyPredictor is a region-aware machine learning system for forecasting **future residential home prices** using historical sales data and property characteristics.

Unlike appraisal tools, it models **price appreciation over time**, explicitly accounting for inflation and regional market behavior.
