# Feature Engineering

## Numeric Features

- Beds
- Baths
- Square footage
- Year built
- Lot size
- Taxes
- ZIP code
- Listing-to-sale ratio

---

## Categorical (One-Hot)

- Home style
- County

---

## Temporal Anchors

- Last sale price
- Days between sales
- Inflation-adjusted baseline

All numeric features are **z-score standardized** before training.
